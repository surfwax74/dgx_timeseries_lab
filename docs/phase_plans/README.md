## Convention

A phase plan in this dir is a commitment: when it lands here, scope and acceptance are locked. Execution-time discoveries get noted as deviations at the bottom of the plan; if they're material, they get a follow-up plan rather than mutating the original.

When a phase completes, mark its row above with **DONE** and the commit SHA. Don't delete the plan — it's the historical record of *why*.
